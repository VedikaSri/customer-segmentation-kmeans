# Customer Segmentation using K-Means Clustering

## Overview
This project applies K-Means Clustering to segment customers into different groups based on their characteristics.

## Objectives
- Perform customer segmentation
- Find the optimal number of clusters using the Elbow Method
- Visualize customer groups
- Analyze clustering results

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Workflow
1. Data Preprocessing
2. Exploratory Data Analysis
3. Elbow Method
4. K-Means Clustering
5. Cluster Visualization

## Results
The model successfully identified distinct customer groups that can help businesses understand customer behavior and improve marketing strategies.
